# Timetracer-Blink
> A Progressive Web Application client for Timetracer-Recall API

A installable PWA for desktop and phone

## Vue dev guide
#### Project setup
```
npm install
```
#### Compiles and hot-reloads for development
```
npm run serve
```
#### Compiles and minifies for production
```
npm run build
```
#### Lints and fixes files
```
npm run lint
```
